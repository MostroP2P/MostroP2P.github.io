Sobre Mostro

Mostro, um projeto de ponta, desenvolve uma abordagem revolucionária para facilitar transações de Bitcoin através da Rede Lightning. O projeto atende à crescente demanda na compra e venda de Bitcoin sem comprometer dados pessoais, apresentando uma solução inovadora para trocas entre pares.

Mostro gerencia comunicações sobre Nostr, tornando-a uma plataforma descentralizada sem nenhum ponto único de falha. Este design melhora a resistência à censura em comparação com outras opções existentes.

Como funciona?

O Monstro atua como um depósito de garantia que permite ao comprador e ao vendedor trocar bitcoin de forma P2P, reduzindo o risco para ambas as partes, ao mesmo tempo em que custodia apenas os Sats pelo tempo mínimo necessário. Ele utiliza um nó da Lightning Network, que cria faturas retidas para os vendedores e paga faturas comuns de Lightning aos compradores.

Funciona dentro do protocolo Nostr, portanto, os compradores e vendedores precisarão dos clientes da Mostro para poder comprar/vender Bitcoin. Os usuários poderão avaliar os Mostros, que competirão para obter mais usuários a fim de sobreviver. Os usuários devem recusar os Mostros ruins, que perderão incentivos para continuar existindo.
